## 📌 Multiplication App

A simple and interactive **Multiplication App** built using **HTML, CSS, and JavaScript**. This project helps users practice multiplication in a fun and engaging way while improving their speed and accuracy.



## 🚀 Features

* ✅ Random multiplication questions
* ✅ Instant feedback on answers
* ✅ Score tracking system
* ✅ Responsive and user-friendly design
* ✅ Lightweight and fast (no frameworks used)


## 🛠️ Technologies Used

* **HTML** – Structure of the app
* **CSS** – Styling and layout
* **JavaScript** – Logic and interactivity


## 🎯 Purpose

This project was created to:

* Practice core **JavaScript concepts** (DOM manipulation, events, logic)
* Build a strong foundation in **frontend development**
* Demonstrate the ability to create functional web apps without libraries
